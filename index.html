Github Pages Boilerplate v2.1.0

Simple boilerplate para comenzar a crear tu sitio web en github.io usando:

  • _ Gulp JS
  • _ Pug
  • _ Stylus

Modo de uso:

  • _ Copia todos los archivos en tu repositorio personal YOUR_USERNAME.github.io, excepto la carpeta oculta .git.
  • _ Inicia el modo desarrollo con el comando: npm run dev
  • _ Solo debes editar el contenido de la carpeta 'src'
  • _ Crea tus carpetas personales dentro de 'views'
  • _ Despues de editar tu contenido usa el comando: npm run build
  • _ Por ultimo sube tus archivos a tu repositorio
