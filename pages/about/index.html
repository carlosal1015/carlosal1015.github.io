A few words about MAD4Jekyll

This site is madly in love with Jekyll builder.


MAD4Jekyll is hosted by Github.

It was built with Jekyll.

It is using MAD4Jekyll theme.

The gallery in the respective page was created with Flexslider.

The photos in the blog posts can be zoomed in on click with the help of Lightbox.

The posts can be written and exported as markdown ready for publication with Dillinger, a useful online editor.

The footnotes popup are rendered with Bigfoot script.

All the commits were performed with the help of Source Tree app.
