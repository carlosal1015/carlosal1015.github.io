Welcome to MADness!

'MAD4Jekyll' is a theme for Jekyll.


It is based on Clean Blog theme by Start Bootstrap.


Features:

-- Integrated slider with captions from Flexslider.

-- Back To Top arrow.

-- A separate section for 'blogging'.

-- A separate section for 'galleries', beside the featured one at the homepage.
